Project Plan

Site Name: Movie Night

I'm calling this site 'Movie Night' because I want a site to support my regular movie nights.

Site Purpose:

It's purpose is to provide information that is helpful for movie night. That includes a calendar of upcoming movie nights, a list of proposed moves to watch and a way to filter them.

Scenarios:

  1. Are we having movie night this week?
  2. What movie should we watch?

Color Schema:

  • Background Color #2F4F4F
  • Text Color #FFFAF0
  • Accents Color #DAA520
  • Links Color #B0C4DE

Typography

Cinzel

I will be using Cinzel for the headings. It has an older cinimatic feel.

Marriweather

I'm going to use Marrieweather for the main text. I feel like it's a very readable font, while still being interesting.
